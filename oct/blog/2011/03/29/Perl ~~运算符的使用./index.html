Perl ~~运算符的使用.

Mar 29th, 2011

之前在锚定的时候遇到过这个符号.和=~是相反的意思.就是无匹配为真…今天在看Effective Perl的时候发现这个东西还真是有用… 首先.当我们在检查hash数组key的存在与否和数组是否存在时使用的是:

my %hash = (
         name => 'wxm',
         score => 99,
);
if( exists($hash{name})){
      print "Exists\n";
}
my @array = (1, 2);
if( grep{ $_ == 2} @array) {
     print "Exists\n";
}

但是这样是很麻烦的.无论你是通过循环还是通过grep等方式进行判断.都是比较繁琐的..所以我们可以使用~~操作符.

print "Exists\n" if 'name'~~%hash;
print "Exists\n" if 2~~@array;

这样就相对简单了. 当然还可以直接比较数组.

# 比较数组.
print "OK" if @arrayA ~~ @arrayB;
# 比较哈希
print "OK" if %hashA ~~ @hashB;
