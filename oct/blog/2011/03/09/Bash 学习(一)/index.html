Bash 学习(一)

Mar 9th, 2011

昨天的信息安全导论.课挺无聊的.就去图书馆随便拽了本书.php|perl|python的进一步需要靠自己了.因为图书资料没什么了.都大同小异.这也是自己第一次对3P产生犹豫.不管怎样.努力.努力.努力! 借了本bash.之前学过.所以浏览的很快.不过期间还是发现一堆不会的问题.就昨天做的笔记记录下吧. shell是UNIX的用户接口. *.首先是查看bash的版本.这个还是很重要的.毕竟有什么问题求助的话.版本是一个重要的信息.

echo $BASH_VERSION

.然后说下shell的环境.首先.用户开启shell时.系统先读取/etc/profile里面的脚本.然后再到用户的家目录下读取~/.bash_profile,~/.bashrc等信息. .然后就是bash的基本通配符.

?    任意单个字符
*    任意字符串
[]   集合中的任意字符
[!]  非集合中的任意字符
{}   任意字符串.

*.接下来就是shell的基本编程.

# 函数可按如下格式定义
function funcName{
       # Code Here
}
# 或者
funcName(){
      # Code Here
}
# 在函数内部通过$1,$2,$*,$@等方式来读取参数.$#来读取参数个数.

在shell中键入一个命令时,先读取别名,然后是关键字,然后是函数,然后是内置命令,然后才是PATH下的可执行程序.可以通过builtin来执行内建命令. 然后就是字符串的操作语法.

${var:-string}     #var为null时.返回string
${var:=string}    #var为null时.var=string
${var:?message} #var为null时.返回message并终止脚本.
${var:+string}    #var不为null时.返回string
${var:offset:offset} #截取变量值.

然后是模式匹配.

${var#pattern}      #从头匹配.删除最短的匹配.然后返回剩余部分
${var##pattern}    #从头匹配.删除最长的匹配.然后返回剩余部分
${var%pattern}     # 从尾匹配.删除最短的匹配.然后返回剩余部分
${var%%pattern}   #从尾匹配.删除最长的匹配.然后返回剩余部分
${var/pattern/string}   #替换.只替换第一个匹配
${var//pattern/string} #替换.替换所有的匹配

*.然后是流程控制. 先是if语句.

if command then
      # Code Here
elif command then
     # Code Here
else
     # Code Here
fi
# 当条件不是command而是简单测试语句时.可以test 测试语句.
# 为了简便.用[];包含起来.
if [ statement ]; then
    # Code Here
elif [ statement ]; then
    # Code Here
else
    # Code Here
fi
# 注意[]左右两端的空格.必须有

然后需要注意的一点是.

# 字符串比较
str1=str2    #str1 匹配 str2
str1!=str2   #str1 不匹配 str2
-n str         #str长度大于0
-z str         #str长度为0
# 数值比较.
-lt             #小于
-gt            #大于
-ne           #不等于
# 文件测试
-d file      # 存在且为一个目录
-e file      # 存在
-f file      # 存在且为一个正规文件
-r file      # 对file有读权限
-s file     # 文件存在且为空
-w file    # 对file有写权限
-x file     # 对file有执行权限.
-O file    # 你是file的所有者
file1 -nt file2 # file1比file2新
file1 -ot file2 # file1比file2旧
# 新旧比较的是修改时间.'

然后就是for循环.

for name in list; do
       #Code Here
done

然后是case的语法.

case expression in
     pattern1 ) statement ;;
     pattern2 ) statement ;;
     * ) statement ;;
esac

然后是select语法.

select name in list; do
      # Code Here
done

例子:

OPTION='apple:orange'
IFS=:
select choice in $OPTION; do
      case $choice in
            'apple'  ) echo 'apple'   ;;
            'orange') echo 'orange' ;;
      esac
done

while和untile语法.

while [condition]; do
     # Code Here
done
