Bash学习(二).参数捕获

Mar 9th, 2011

命令行参数的捕获方法有好多思路.不过因为不同的用户有不同的参数习惯.所以最好还是通过系统内置的getopts来获取吧. 下面是我写的.通过循环和shift来实现.不过无法解析-ab这种参数.

#!/bin/bash
declare -i argvCount=$#
echo $argvCount
until [ $argvCount -eq 0 ]; do
    case $1 in
        'apple' ) echo 'apple' ;;
        'orange' ) echo 'orange' ;;
    esac
    shift
    argvCount=$argvCount-1
done

接下来是由getopts来实现的版本.

#!/bin/bash
while getopts "ao" opt; do
    case $opt in
        a) echo 'apple'  ;;
        o) echo 'orange' ;;
    esac
done
