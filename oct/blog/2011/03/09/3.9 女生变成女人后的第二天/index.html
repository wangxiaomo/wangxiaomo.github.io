3.9 女生变成女人后的第二天

Mar 9th, 2011

今天阿娇姐好不容易问个问题.结果因为我正在看bash.内容都混了.太悲剧了. 就是一个<a[>]+>的匹配问题.

#!/usr/bin/perl
use warnings;
use feature ":5.10";

$str1 = 'asd>';
say $1 if $str1 =~ /(]+>)/i;

$str2 = 'asd>';
say $1 if $str2 =~ /(]+>)/i;

$str3 = 'asd>';
# 你写的.如果出现空白符或者其他符号.无法匹配
say $1 if $str3 =~ /()/i;
say $1 if $str3 =~ /()/i;
__END__
上面的匹配都是贪婪匹配.
