Perl添加[模块]引用地址

Mar 30th, 2011

经常写Perl的会发现总会被提醒这样那样的@INC错误.其实就是包含路径错误.一般情况下如果做了正确的处理是不该出现这样的问题的.所以我还是把正确的处理方式说下.以免以后继续悲剧了. 首先.我们可以通过unshift把地址添加到@INC中.

BEGIN{
     unshift @INC,"/usr/share/perl";
}

或者就是.

use lib "/usr/share/perl";
#添加多行地址.
use lib qw(
      /usr/share/perl
      /usr/perl
);
