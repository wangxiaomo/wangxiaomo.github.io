以上是没有参数的请求.接下来讲带有参数的请求.

XMLRPC请求中带有的参数必须指定XMLRPC类型.这些参数可以通过PHP本地变量的方法进行传递.

PHP变量类型          XMLRPC变量类型
integer                        int
double                        double
boolean                      boolean
string                         string
array                           array
associativearray          struct
object                         array
