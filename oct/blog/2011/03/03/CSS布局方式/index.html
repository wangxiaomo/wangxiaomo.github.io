CSS布局方式

Mar 3rd, 2011

今天在ub下的bluefish下面写css.老是发现布局错误.我就无语了.算了半天px.还尝试加了N多的clear.还是不行.后来就果断切系统到dreamweaver下来写css了.毕竟边写边测试.比较爽. 为了排除浏览器不通所造成的影响.一般情况下先把margin和padding设置为0.

* {
    margin:0;
    padding:0;
}

然后就是为了使页面适应不同分辨率的显示器.分配一个页面div.

#page {
    width:960px;/*大小看情况决定.美观就好*/
    margin:0 auto;
}

然后先说今天的第一个布局.就是一行两列形式.




     
 

     
 

最让我奇怪的是.一开始测试了N回都不行.然后就突然成功了.float脱离文档流吗?哎.搞不清楚了.完了再探索吧.然后我测试了下向同一方向的浮动.成功了.没啥可说的.那么一行3列也就可以通过float来做了.可是为嘛我在ub下没成功呢?
