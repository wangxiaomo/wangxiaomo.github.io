Python Sub中使用捕获到的值

Sep 1st, 2011

刚才看郭靖写代码的时候突然想到了这个,但是查了半天也没找到相关的文档.但是下面这个demo成功了.lambda果真是个变态级别的东西

import re
s = "abc1abc"
print re.sub(r'(\d+)', lambda m: "f"+m.group(0)+"f", s)
