4.28. Ub11.04 发布

Apr 28th, 2011

昨晚看小说看的太HIGH了. 早上起来就7:50了. 赶紧洗涮了就飞奔. 到了教室才发现忘了带眼镜了. 所以后半节课就果断翘课了. 昨晚清哥写代码写到很晚. 抽住了. 有女生求他写代码 . 他就写的更入迷了似的. 上学期课设的时候我做过这个. 不过代码丢了. 大概印象还是有的. 现在就把代码确认下. 大概思路先说下:

首先connect默认是阻塞连接.阻塞连接时会持续请求连接.
大概30秒或者更长的时间才会返回.
当我们做端口扫描的时候完全不可能等待那么长时间.
所以我们需要先把socket设置成非阻塞的.
那么connect就会瞬间返回.[我们可以设置connect时间.].
然后我们通过select来判断套接字是否可读写.
这样就可以了.

下面看代码.

#include "stdio.h"                                                  
#include "winsock2.h"                                                
#pragma comment(lib, "ws2_32.lib")             
                                                                                                                             
#define IP_ADDRESS "127.0.0.1"       
#define PORT 8080          
                                                                                                                             
int main(void){     
        WSADATA wsaData;  
        WSAStartup(MAKEWORD(2,0), &wsaData); 
        SOCKET client = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP); 
        //int timeOut = 2000; // 2 seconds    
        //setsockopt(client, SOL_SOCKET, SO_SNDTIMEO, (char *)&timeOut, sizeof(timeOut));    
        //setsockopt(client, SOL_SOCKET, SO_RCVTIMEO, (char *)&timeOut, sizeof(timeOut));
        unsigned long ul = 1;   
        ioctlsocket(client, FIONBIO, (unsigned long *)&ul);   
        SOCKADDR_IN saddr;         
        saddr.sin_family = AF_INET; 
        saddr.sin_addr.s_addr = inet_addr(IP_ADDRESS);  
        saddr.sin_port = htons(PORT);    
        connect(client, (SOCKADDR *)&saddr, sizeof(SOCKADDR)); 
        struct timeval timeout;
        fd_set r;        
        FD_ZERO(&r);      
        FD_SET(client, &r);   
        timeout.tv_sec = 2;  
        timeout.tv_usec = 0;    
        int ret = select(0, 0, &r, 0, &timeout); 
        // TODO with ret  
        if(ret > 0){     
                // Socket Open    
        }else{                   
                // Socket Close
        }
        closesocket(client);
        WSACleanup();
        return 0;
}

That’s All.`
