Fedora_14_Padre使用

Apr 5th, 2011

Fedora14下默认是没有Padre的.所以我从库里搜索了下.

yum search padre

搜索出来了好多关于padre的东西.但是我并不确定装哪个.因为yum不像aptitude的search.

aptitude search padre
# i padre      -Perl Application Develpment
yum search padre
#perl-padre  Perl Application ...类似的一堆东西.
#反正我不知道到底起作用的是哪个.
#所以我也就没管.直接yum install padre了.
#嘿.竟然成功了.

之后在终端提示缺少so文件.libwx_gtk2u_aui 然后google了半天.没什么有用的资料. 然后我就查看了下gtk库

yum search gtk

发现gtk-devel没有安装.果断安装了.然后就可以正常使用了`
