Perl_CGI编程.

Apr 1st, 2011

Perl CGI编程虽然是一个比较古老的东西.但他的实际应用还是很多的. 使用CGI编程.

use CGI qw(:standard);

使用如上模式可以直接使用header.start_html等函数.如果不包含qw的话.必须实例化CGI. 然后就类似于正常编程了.毕竟熟悉HTML/CSS的话还是自己通过如下写比较舒服.

print <<"HTML";

HTML

然后就是关于get/post的元素捕获了.就是通过param()函数来实现.

# 检测是否有get/post信息.
if(param()){
    # Code Here
    # get/post data: name=wxm&passwd=123
    # $name = param('name');
    # $passwd = param('passwd');
}else{
    # ...
}

同param可以通过cookie函数来读取或者设置cookie. 至于其他的就靠自己发挥了. 对了.还有个调试的技巧. 1.写完CGI后先perl -c检查语法. 2.使用CGI::Carp模块.

use CGI qw(:standard);
use CGI::Carp qw(fatalsToBrowser);
# 可以输出到浏览器use CGI::Carp qw(carpout);
open LOG,'>','/var/log/cgi-error';
carpout(LOG);
# 可以输出到文件.
