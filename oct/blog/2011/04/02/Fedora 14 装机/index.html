Fedora 14 装机

Apr 2nd, 2011

我一定是疯了.不然为啥用ubuntu用的好好的要装fedora啊.主要是最近有点闲….不过fedora还真给力.还真没我想象中那么好搞… 直接跳入正题:

yum update 
# wait about 5 min
yum install vim

vi .bashrc
vi .vimrc
# both root & user

yum install gcc gcc-c++ *aclocal
# install build-essential

# install Chinese into the OS
yum search chinese | grep ^kde
yum install kde-l10n-Chinese.noarch

# install fcitx
yum search fcitx
yum install fcitx.i686
yum install gtk2-immodule-xim 

# Download flash player
mv libflashplayer.so /usr/lib/mozilla/plugins/
chmod 777 libflashplayer.so
