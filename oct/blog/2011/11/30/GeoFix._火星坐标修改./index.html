GeoFix._火星坐标修改.

Nov 30th, 2011

今天gps传回的数据在google earth中有很大的误差.老周说是经纬度要校验后再显示,突然就想起了前阵子做looking4u时候看到的火星坐标的问题.饿.出了这个问题没有第一时间想起来这个问题,my fault. 火星坐标是为了保护国家安全信息的…火星坐标与标准地理坐标的转化也比较麻烦. 据CSDN说,转化过程是这个样子: 火星坐标除以∏，再开8次方，再以e为底求一次对数，再展开为傅立叶级数，取其中奇数项常量做卷积，可得相应地球坐标。 不过我觉得这个比较不靠谱. 火星坐标的转化公式也是每过一段时间就变化的.所以需要找一个简单而比较犀利的方法. 方法一.http://www.anttna.com/goffset/goffset1.php?lat=纠偏前的纬度值&lon=纠偏前的经度值. 恩.这个方法比较简单. 方法二.先下载两个xml文件.SXML.xml和DXML.xml文件.然后运行如下代码:

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Xml;

namespace GeoFix{
    class GeoFix{
        public static string sxml = "sxml.xml";
        public static string dxml = "dxml.xml";


        // Geo Fix. 
        // 修正火星坐标.

        public static double[] fix(double lat, double lng){
            Double[] ret = refix(lat, lng, 0);
            if(ret[0] == 0.0 && ret[1] == 0.0){
                ret = refix(lat, lng, 1);
            }
            return ret;
        }

        public static double[] refix(double lat, double lng, int type){
            Double[] ret = new Double[2]{0.0, 0.0};
            XmlDocument xmlDoc = new XmlDocument();
            if(type == 0){
                xmlDoc.Load(sxml);
            }else{
                xmlDoc.Load(dxml);
            }
            XmlNodeList nodes = xmlDoc.GetElementsByTagName("data");
            foreach(XmlNode node in nodes){
                XmlNodeList childNodes = node.ChildNodes;
                double xla = double.Parse(childNodes[0].InnerText);
                double xlo = double.Parse(childNodes[1].InnerText);
                double dla = double.Parse(childNodes[2].InnerText);
                double dlo = double.Parse(childNodes[3].InnerText);
                double la = double.Parse(childNodes[4].InnerText);
                double lo = double.Parse(childNodes[5].InnerText);
                if(xlalat && xlolng){
                    ret[0] = lat-la;
                    ret[1] = lng-lo;
                    break;
                }
            }
            return ret;
        }
    }
}

两个方法都可以进行校验,不过效果.恩..我觉得一般…有待后续改进..
