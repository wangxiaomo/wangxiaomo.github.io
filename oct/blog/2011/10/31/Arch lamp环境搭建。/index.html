Arch Lamp环境搭建。

Oct 31st, 2011

环境需要，搭建lamp.arch的lamp环境搭建起来还真是比ubuntu，fedora之类的费劲多了，不过配置思路却很明确，对amp之间的依赖关系的学习有一定的帮助。 废话不多说。

pacman -S apache mysql php

安装amp。重点是接下来的配置。 apache和php.

vi /etc/httpd/conf/httpd.conf
#
# 添加配置文件
# Include /etc/httpd/conf/extra/php5_module.conf
# 注释mod_unique_id模块。
# 添加php模块。
# LoadModule php5_module modules/libphp5.so
# 添加类型
# AddType application/x-httpd-php .php
# AddType application/x-httpd-php-source .phps

重启服务器。写入phpinfo。测试成功。 php和mysql。

vi /etc/php/php.ini
# 注释mysql.so

重启服务器。

cd /srv/http
php phpinfo.php | grep mysql

添加到daemon中。

vi /etc/rc.conf
# mysqld httpd
