Sql_server_启用sa用户

Oct 6th, 2011

sql server2005安装完默认是windows验证模式.此时sa用户没有启用.这时,我们把数据库服务器改为混合认证方式.并重设sa密码.启用sa用户即可. 具体参考http://technet.microsoft.com/zh-cn/library/ms188670.aspx
