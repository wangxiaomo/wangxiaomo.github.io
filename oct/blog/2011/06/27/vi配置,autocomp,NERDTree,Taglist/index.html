vi配置,autocomp,NERDTree,Taglist

Jun 27th, 2011

今天等待android sdk的下载,就把vi配置了下.目前只有两个基本需求: 1.文件管理.. NERDTree 2.自动补全.autocomplPop. 3.生成变量类信息的Taglist,需要ctag的配合. vi的插件可以放在很多地方,我们可以通过

set runtimepath?

来查看可放置的位置.. 假设我们要把插件放置再~/.vim下.我们一般要建立以下文件夹.

+~/.vim
+-----autoload
+-----plugin
+-----doc

然后我们在vim.org上下好上面两个插件,然后放在相应的目录下就可以了… 启动NERDTree.只需要键入

:NERDTree

自动补全的插件已经autoload了.` 然后最重要的就是我的.vimrc的配置.


set tabstop=4
set sm
set number
syntax on
colorscheme desert

"Set to Auto read when a file is changed from the outside
set autoread
"Set Magic on for regex
set magic
"显示光标位置.
set ruler
"侦测文件类型
filetype on
"隐藏工具栏
set go=
"设置高亮搜索.
set hlsearch
set showmatch

"在左侧窗口显示TlistOpen
let Tlist_Use_Left_Window=1
let Tlist_Show_One_File=1
let Tlist_Exit_OnlyWindow=1
let Tlist_Show_Menu=1
"let Tlist_Auto_Open=1
"编码自动识别.
set fencs=utf-8,cp936
:nmap  :NERDTreeToggle
:nmap  :TlistToggle

let g:winManagerWindowLayout="FileExplorer|TagList"
:nmap  :WMToggle
