Android菜单设计.

Jun 24th, 2011

Android菜单分为选项菜单,上下文菜单和子菜单… 1.选项菜单.选项菜单就是单击设备上Menu弹出的菜单.菜单项最多只能有6个,超过6个第6个会自动显示为”更多”来展开显示. 使用方法:

覆盖Activity的onCreateOptionsMenu(Menu menu)方法创建选项菜单.
覆盖Activity的onOptionsItemSelected(MenuItem item)方法响应选项菜单.

示例:

//菜单的设计不在这里讲述..
package MyFirst.Test;

import android.app.Activity;
import android.os.Bundle;

import android.view.View;
import android.view.Menu;
import android.view.MenuInflater;
import android.view.MenuItem;

public class MyFirstTestActivity extends Activity{
    /** Called when the activity is first created. */
    private MenuInflater mi;
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        // 设置当前布局.
        setContentView(R.layout.main);
        mi = new MenuInflater(this);
    }
    public boolean onCreateOptionsMenu(Menu menu){
        mi.inflate(R.menu.menu, menu);
        return true;
    }
    public boolean onOptionsItemSelected(MenuItem item){
        CharSequence title = item.getTitle();
        if(title.equals("Start")){
            setTitle("Start");
        }else{
            setTitle("Stop");
        }
        return true;
    }
}

当然,如果不想通过xml文件建立菜单的话,也可以动态创建.

private static final int ITEM1 = menu.First;
private static final int ITEM2 = menu.First+1;
public boolean onCreateOptionsMenu(Menu menu){
    menu.add(0, ITEM1, 0, "Start");
    menu.add(0, ITEM2, 0, "Stop");
    return true;
}
public boolean onOptionsItemSelected(MenuItem item){
    switch(item.getItemId()){
        case ITEM1:
            setTitle("Start");
            break;
        case ITEM2:
            setTitle("Stop");
            break;
     }
}

2.上下文菜单..当用户长时间案件不放时,弹出的菜单就称为上下文菜单.

覆盖Activity的onCreateContextMenu()方法,调用Menu的add方法添加菜单项MenuItem
覆盖Activity的onContextItemSelected()方法,响应事件.
调用registerForContextMenu()方法,为视图注册上下文菜单.

示例.

//需要绑定到TextView上.
import android.widget.TextView;
import android.view.Menu;
import android.view.MenuItem;
import android.view.ContextMenu;
import android.view.ContextMenu.ContextMenuInfo;
private static final int ITEM1 = menu.First;
private static final int ITEM2 = menu.First+1;
public void onCreateContextMenu(ContextMenu menu,View v,
                                            ContextMenuInfo menuInfo){
    menu.add(0, ITEM1, 0, "Start");
    menu.add(0, ITEM2, 0, "Stop");
}
public boolean onContextItemSelected(MenuItem item){
    switch(item.getItemId()){
        case ITEM1:
            setTitle("Start");
            break;
        case ITEM2:
            setTitle("Stop");
            break;
     }
}

注意,上下文菜单的重载函数与选项菜单的重载函数返回值不完全一样.. 3.子菜单.

覆盖Activity的onCreateOptionsMenu()方法,调用Menu的addSubMenu方法添加子菜单.
调用SubMenu的add方法.添加子菜单.
调用onOptionsItemSelected()方法,响应事件.

示例:

// import android.view.SubMenu;
private static final int ITEM1 = menu.First;
private static final int ITEM2 = menu.First+1;
public boolean onCreateOptionsMenu(Menu menu){
    SubMenu file = menu.addSubMenu("File");
    file.add(0,ITEM1,0,"Start");
    file.add(0,ITEM2,0,"Stop");
    return true;
}
public boolean onOptionsItemSelected(MenuItem item){
    switch(item.getItemId()){
        case ITEM1:
            setTitle("Start");
            break;
        case ITEM2:
            setTitle("Stop");
            break;
     }
}
