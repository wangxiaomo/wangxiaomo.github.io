Ubuntu Awesome完美结合

Jun 5th, 2011

之前一直没有把awesome用到极致,虽然能配合idesk使用.但是感觉还是有很多地方有点不爽. 今天可算是全部搞定了. 主要就是需要把gnome-session载入进去,fluxbox也可以使用相同方法. awesome的载入比较麻烦,所以这里就写下awesome的配置.

-- AutoRun
autorun = "true"
autorunApps = {"gnome-session"}
if autorun then
    for app=1,# autorunApps do
        awful.util.spawn(autorunApps[app])
    end
end
