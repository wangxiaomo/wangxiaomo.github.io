MFC AfxBeginThread使用.

Jun 11th, 2011

课设,饿,真的没有投入太多的精力,倒是让卓子写了不少,卓子会不会觉得我很不靠谱啊…哎.. 我要写一个伪进度条,恩,伪的.网速是未知的,所以进程是不可控的.. 伪进程,需要用多线程. 主线程用CreateProcess来执行CMD命令,然后分出两个线程,一个用来死循环进度条,一个用来设置timer,用来控制超时… BeginThread,_beginthread,AfxBeginThread,方案有很多,我毅然决然的选择了afx的.mfc么.而且,AfxBeginThread大多都是默认参数.我喜欢… 呵呵

UINT __cdecl test1(LPVOID pParam){
     Dialog1 m;
     m.DoModal();
}
UINT __cdecl test2(LPVOID pParam){
     Dialog2 m;
     m.DoModal();
}
//在initdialog中.
CWinThread *t1 = AfxBeginThread(test1, NULL);
CWinThread *t2 = AfxBeginThread(test2, NULL);
//在某一响应中.
TerminateThread(t1->m_hThread, 0);

就这么简单`
