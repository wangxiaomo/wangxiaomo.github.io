Ajax_w3school教程

Feb 28th, 2011

学习了xajax的使用.进一步学习下ajax吧.毕竟xajax的缺点还是有的.最好多了解点. AJAX指异步Javascript及XML.AJAX不是一种语法.而是一种模式. 通过AJAX,你的javascript可使用javascript的XMLHttpRequest对象来直接与服务器进行通信,通过这个对象,你的javascript可以在不重载页面的情况与Web服务器交换数据. AJAX在浏览器与Web服务器之间使用异步数据传输(HTTP请求),这样就可以使网页从服务器请求少量的信息,而不是整个页面. 在传充的数据传输过程中(GET或者POST),用户需要点击提交表单,然后等待服务器响应,然后一张新的页面会被加载.由于每当用户提交输入后服务器都会返回一张新的页面,所以应用程序变得运行缓慢,且越来越不友好.通过AJAX,你可以直接与服务器进行通信,用户可以停留在一个页面上,他不会注意到脚本在后台的请求过程. Google建议使用XMLHttpRequest对象创建一种动态性极强的web界面.就是google的搜索建议.自此.AJAX被广泛使用. XMLHttpRequest需要下列浏览器的支持:IE5.0+,Safari 1.2,Mozilla 1.0/Firefox,Opera8+,Netscape 7.

为了理解AJAX的工作原理.我们将创建一个AJAX小型应用. 首先.我们需要编写一个简单的表单.

AJAX的要点是XMLHttpRequest对象.不同浏览器创建该对象的方法是不同的.IE浏览器使用ActiveObject而其他浏览器使用名为XMLHttpRequest的Javascript内建对象.

声明一个xmlHttp变量存储XMLHttpRequest对象.然后通过异常方式创建通用AJAX对象. 在向服务器发送数据之前.我们需要了解下XMLHttpRequest对象的三个重要属性. onreadystatechange属性存有处理服务器响应的函数.

readyState属性存有服务器响应的状态信息.每当readyState改变时,onreadystatechange函数就会被执行. readyState可能的值: 0 请求未初始化(在调用open()之前) 1 请求已提出(在调用send()之前) 2 请求已发送 3 请求处理中 4 请求已完成 所以我们要在onreadystatechange函数中添加一条if语句,来测试我们的响应是否已经完成.

responseText属性来取回服务器返回的数据.

以上便是接受服务器响应并做出处理的代码.接下来讲怎么向服务器发送数据. 要想把数据发送到服务器.需要使用open方法和send方法. open方法需要三个参数.第一个定义发送请求所使用的方法(GET或POST).第二个定义服务器端脚本的URL.第三个规定对请求进行异步处理. send方法可将请求发送到服务器. 假设HTML文件和ASP文件位于相同的目录.

<%
response.expires = -1
dim a(30)
'用名字为数组赋值.
a(1) = "Anna"
'.....
q = ucase(request.querystring('q'))
if len(q)>0 then
    hint=""
    for i=1 to 30
        if q=ucase(mid(a(i),1,len(q))) then
            if hint="" then
                hint=a(i)
            else
                hint=hint&" , "&(i)
            end if
        end if
    next
end if
if hint="" then
    response.write("no suggestion")
else
    response.write(hint)
end if
%>
