XAJAX七步曲中的一些问题

Feb 27th, 2011

今天实在不想弄delphi了.就整了一会xajax.以前看过教程.当时觉得很简单.就没代码测试.今天测试了一下还真发现了不少问题. xajax以易用性出名.官方文档中的七步曲也写的很不错.唯一失败的就是你按照其步骤做下来也无法得到正确的结果. 首先.版本问题.我果断下了beta版本.然后在linux/nginx下测试.无论是它压缩包中给的例子还是自己写的东西.都是错误的.查看log.发现没什么特别的提示.网上说打开debug开关.

debugOn();
?>

我试了下.不行.在日志中提示无法找到debugOn这个方法.具体怎么解决.等有时间再说. 然后我查阅了半天资料.也没发现解决方法.甚至看到了不少人在评论中说放弃了xajax.哎.是不是因为我当初这个平台没有搭好啊.所以我果断切到了windows.用wamp测试下. 这次我学乖了.我下的是stable版本.然后先运行了它的例子.但是还是错误的.Deprecated错误.然后在源码中加入了@抑制符.还是不行.然后上网查了查.发现stable(0.5)不支持PHP5.所以又下了beta(0.6). 例子倒是没出现什么问题了.然后我就自己写了一个脚本测试.

assign('content', 'innerHTML', $str);
    return $obj;
}

$xajax = new xajax();
$xajax->register(XAJAX_FUNCTION,'test');
$xajax->processRequest();
?>
