编码终极解决方案

May 29th, 2011

一直纠结于编码, 今天终于找到了点门路… 字符串在Perl或者Python内部都是unicode的形式,因此,在做编码转化的时候通常以unicode为中间编码,然后通过encode来转化为特定的编码.即先探测字符串编码(chardet,Encode::Guess),然后decode掉,成为unicode编码,然后再通过encode来转化编码…


然后,要注意的是,代码中的字符串与文件的编码一致.所以说Python中要想使字符串为unicode而不受文件编码的影响,需要用u”来定义. 如果对unicode进行解码则会发生错误,所以我们需要先判断.以Python举例/

import chardet
s = chardet.detect(str)
if s.encoding == "utf-8":
    pass

isinstance(s, unicode)

我们可以通过sys模块来获得系统默认编码.

import sys
print sys.getdefaultencoding()
