Python Logging模块

May 25th, 2011

python2和python3之间的差别还是比较大的. 不过貌似现在还是2应用的更广泛些 . 不扯淡了. 直接说logging模块. 之前看过的例子,因为各种handler,所以不是很懂. 其实一个简单的日志, 很容易就可以做出来(不包含handler)

#!/usr/bin/python -tt
# -*- coding: utf-8 -*-

import logging
log = logging.basicConfig(
    filename = "err.log",
    format = "%{levelname}-10s %{asctime}s %{message}s",
    level = logging.DEBUG
)

logger = logging.getLogger(log)
logger.info("Error")
