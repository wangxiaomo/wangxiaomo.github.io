Pygame编程

May 7th, 2011

本来我是想下一个关于python的视频教学的.本想稍微复习下.结果误打误撞的下到了pygame的视频教学.那我就索性学习下吧 首先是pygame.可以上google找到其官网.我直接懒得上官网下文档了.索性看英文视频算了.

aptitude search pygame
aptitude install python-pygame

安装好pygame后我们就可以import查看version.来观察是否安装成功.

import pygame
print pygame.ver
#1.9.1release

接下来我们写一个简单的程序.

#!/usr/bin/python
import pygame, sys
from pygame.locals import *

#初始化
pygame.init()
screen = pygame.display.set_mode((320,240), 0, 32)
# 第一个参数是大小.第三个参数是色彩

# 载入图片
bg_pic = 'bg.xcf'
mouse_pic = 'mouse.xcf'
background = pygame.image.load(bg_pic).convert()
mouseCur = pygame.image.load(mouse_pic).convert_alpha()
# convert转化为普通图片.convert_alpha转化为分层图片.即背景透明.

# 进入MainLoop显示界面.
while True:
   # 捕捉事件
   for event in pygame.event.get():
         if event.type == QUIT:
             pygame.quit()
             sys.exit()
   # 设置背景
   screen.blit(background, (0,0))
   
   # 设置鼠标所指为图片中心
   x,y = pygame.mouse.get_pos()
   x -= background.get_width()/2
   y -= mouseCur.get_height()/2
   
   screen.blit(mouseCur, (x, y))
   
   pygame.display.update()

接下来我们探讨下如何进一步捕捉键盘事件.

for event in pygame.event.get():
    if event.type == QUIT:
        # 退出事件.
        pygame.quit()
        sys.exit()
    elif event.type == KEYDOWN:
        # 键盘按下事件
        if event.key == K_UP:
        # 按下上键. 键盘对应键为K_UP,K_DOWN,K_x等.

然后我们开始学习Pygame的做图.

pygame.lock()
# 做图
pygame.unlock()

pygame的做图包含在lock之间.

pygame.draw.rect(screen, (140,240,130), Rect((100,100),(130,170)))

如上在100,100位置上做了130x170大小的矩形,颜色由RGB(140,240,130)填充. 接下来是pygame画多边形.

points = [(20,100),(140,140),(110,30)]
color = (255,255,0)
# draw加到lock后.
pygame.draw.polygon(screen, color, points)

接下来是画圆.

color = (200, 180, 100)
position = (300, 176)
r = (50)
pygame.draw.circle(screen, color, position, r)
