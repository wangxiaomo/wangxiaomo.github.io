关于这几天的一些想法

May 6th, 2011

好久没有碰技术了.不知道为啥.热情提不起来了.真的现实与我当初的想法出现了冲突矛盾的时候,我会选择暂时逃避一段时间.但想到自己的颓废时候又更是一种无形的压力. 不是因为魔兽世界.不是因为touch.只是因为自己的心结. 不废话了.说那么多.who care. 今天立夏了.也是我连续的第二个通宵.早上下雨了.在睡梦中听雨.感觉很好. 前天晚上把实验室的机子装上了win7+fedora.xp真的不行.与linux共存互访后,偶尔会出现无法修复的BUG.所以还是到win7吧.还有一个原因是因为通宵挂机下载的时候linux的下载工具真的不是很给力. 说的fedora.这次不准备用kde了.想朴素点.怀旧点.就果断使用fluxbox了.

yum update
yum install fluxbox

然后注销.选择fluxbox为桌面.然后登录就可以了. 然后就是汉化的问题.之前都是yum install i10n-kde-chinese.这里估计起不了太大作用.也没尝试.就这样了. fluxbox默认是xterm.对字符宽度显示有问题.我也懒得管了.毕竟xterm太丑了.

yum install gnome-terminal

然后就是到~/.fluxbox下进行一些fluxbox的设置. 之后又安装了idesk.这样就初步成型了. ——————————–华丽丽的分割线 今天把<<结网>>看完了.很薄的一本书.但是内容很值得消化.糗事百科创始人的视角很独特很细腻.中间有许多我们需要学习的. 不废话了.最近考试比较多.事也比较多.没有时间可以浪费了. 减肥锻炼也该继续了.touch的话游戏还是少玩点吧.多看看书不错. 实验室的事情该上点心了. 加油.
