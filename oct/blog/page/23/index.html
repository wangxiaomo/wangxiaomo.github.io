Ajax W3school教程

Feb 28th, 2011

学习了xajax的使用.进一步学习下ajax吧.毕竟xajax的缺点还是有的.最好多了解点. AJAX指异步Javascript及XML.AJAX不是一种语法.而是一种模式. 通过AJAX,你的javascript可使用javascript的XMLHttpRequest对象来直接与服务器进行通信,通过这个对象,你的javascript可以在不重载页面的情况与Web服务器交换数据. AJAX在浏览器与Web服务器之间使用异步数据传输(HTTP请求),这样就可以使网页从服务器请求少量的信息,而不是整个页面. 在传充的数据传输过程中(GET或者POST),用户需要点击提交表单,然后等待服务器响应,然后一张新的页面会被加载.由于每当用户提交输入后服务器都会返回一张新的页面,所以应用程序变得运行缓慢,且越来越不友好.通过AJAX,你可以直接与服务器进行通信,用户可以停留在一个页面上,他不会注意到脚本在后台的请求过程. Google建议使用XMLHttpRequest对象创建一种动态性极强的web界面.就是google的搜索建议.自此.AJAX被广泛使用. XMLHttpRequest需要下列浏览器的支持:IE5.0+,Safari 1.2,Mozilla 1.0/Firefox,Opera8+,Netscape 7.

Smarty与PHPLIB的DB类配合使用

Feb 25th, 2011

PHPLib中的db类与Smarty结合. 首先.先介绍下PHPLIB中的db类. DB_Sql的主要成员变量和成员函数: 一.成员变量. 1.$Host:主机名 2.$Database:数据库名称 3.$User:用户名 4.$Password:密码 二.成员函数. 1.connect($Database=”“, $Host=”“, $User=”“, $Password=”“):建立连接.返回连接ID. 2.query($Query_String):进行查询.返回查询资源. 3.free():释放当前查询结果资源. 4.next_record():返回下一条记录集. 5.num_rows():当前查询中的数据行数. 6.f($name):当前字段的值. 其他成员变量及函数就不在此介绍了.具体见PHPLIB文档. 下面见一个示例:

Host = 'localhost';
//数据库名称
$db->Database = 'news';
//用户名和密码
$db->User = 'root';
$db->Password = '';

//数据库连接
$db->connect();
//定义SQL语句.
$strQuery = 'SELECT * FROM table ';
$db->query($strQuery);
$i = NUM;
while($db->next_record() && $i>0){
    $array[] = array('NewsID'=>$db->f('iNewsID'),
                    'NewsTitle'=>$db->f('vcNewsTitle'));
    $i--;
}
$smarty->assign('NEWS', $array);
unset($array);
$db->free();
?>

这样就搞定了. 有时间一定把PHPLIB也学学.

PHPChina Smarty教程

Feb 25th, 2011

最近在看PHP China写的Smarty教程.感觉Smarty是一个很给力的东西.这几天有点浮躁.去网吧去多了.静下心来好好看看书吧. Smarty Smarty的优点需要记好.便于说服别人.哈哈 速度->采用Smarty能得到较大的速度提升.主要是得益于编译和缓存技术.采用Smarty编写的程序在第一次运行时编译成一个非模板技术的PHP文件.这个文件采用PHP和HTML混合的方式,在下一次访问模板时讲直接请求于这个文件.而不再需要从模板重新编译(在源文件没有改动的情况下).而缓存是把相对稳定的内容以静态形式作为缓存输出.相当于调用一个静态的HTML文件.不需要解析时间. Smarty的以上特点决定其不能使用在频繁更新的地方. 一般情况下.用户在每次访问PHP程序时,都会建立新的数据库连接并重新获取一次数据,再经过操作处理形成HTML等响应给用户.这是由于HTTP的无状态性造成的.对于每次页面的请求,都要重复地执行相同的操作,而不论数据是否被更改,但有些信息经常是不变的.如果不想每次都重复地执行某些操作,就可以在第一次访问PHP程序时,把动态获取的HTML输出缓存为静态页面.在以后每次请求该页面时,直接去读取缓存,而不许要不必要的开销. Smarty的编译就是一个模板转换的过程.Smarty的编译减少了模板转化的开销.


Smarty的标准模板文件是tpl.不过为了美工方便.也可以写成html.
