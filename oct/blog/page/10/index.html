        findViews();

        btn_yes.setOnClickListener(new OnClickListener(){
            public void onClick(View v){
                Notification n = new Notification();
                n.icon = R.drawable.icon;
                n.tickerText = "Hello";

                nm.notify(1, n);
            }
        });
        btn_no.setOnClickListener(new OnClickListener(){
            public void onClick(View v){
                nm.cancel(1);
            }
        });
     }
     public void findViews(){
        ;
     }
}

Android系统提供了4种默认的对话框:警告对话框AlertDialog,进度对话框ProgressDialog,日期选择对话框DatePickerDialog和时间选择对话框TimePickerDialog.

获得AlertDialog的静态内部类Builder对象,由该类来创建对话框.
通过Builder来设置对话框的标题,按钮,以及将要响应的事件.
调用Builder的create()方法创建对话框.
抵用AlertDialog的show()方法来显示对话框.

        btn.setOnClickListener(new OnClickListener(){
            public void onClick(View v){
                builder.setMessage("Yes?").
                setPositiveButton("是",new DialogInterface.OnClickListener() {
                    public void onClick(DialogInterface dialog, int which) {

                    }

这个方法很不灵活,主要是对话框默认没有绑定父窗口,所以Android管理起来比较不合理.虽然可以通过setOwnerActivity(Activity)来设定,但是还是很复杂… 所以我们用另外一个策略.. 对话框经常作为Activity的一部分来创建和显示,所以我们应该从protected Dialog Activity.onCreateDialog(int id)回调方法里创建对话框,当你使用这个回调函数的时候,Android系统会有效的设置这个Activity为每个对话框的所有者,从而自动管理每个对话框的状态并挂靠到Activity上.

Jun 24th, 2011

Android菜单分为选项菜单,上下文菜单和子菜单… 1.选项菜单.选项菜单就是单击设备上Menu弹出的菜单.菜单项最多只能有6个,超过6个第6个会自动显示为”更多”来展开显示. 使用方法:

覆盖Activity的onCreateOptionsMenu(Menu menu)方法创建选项菜单.
覆盖Activity的onOptionsItemSelected(MenuItem item)方法响应选项菜单.

示例:

//菜单的设计不在这里讲述..
package MyFirst.Test;

import android.app.Activity;
import android.os.Bundle;

import android.view.View;
import android.view.Menu;
import android.view.MenuInflater;
import android.view.MenuItem;

public class MyFirstTestActivity extends Activity{
    /** Called when the activity is first created. */
    private MenuInflater mi;
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        // 设置当前布局.
        setContentView(R.layout.main);
        mi = new MenuInflater(this);
    }
    public boolean onCreateOptionsMenu(Menu menu){
        mi.inflate(R.menu.menu, menu);
        return true;
    }
    public boolean onOptionsItemSelected(MenuItem item){
        CharSequence title = item.getTitle();
        if(title.equals("Start")){
            setTitle("Start");
        }else{
            setTitle("Stop");
        }
        return true;
    }
}

当然,如果不想通过xml文件建立菜单的话,也可以动态创建.

private static final int ITEM1 = menu.First;
private static final int ITEM2 = menu.First+1;
public boolean onCreateOptionsMenu(Menu menu){
    menu.add(0, ITEM1, 0, "Start");
    menu.add(0, ITEM2, 0, "Stop");
    return true;
}
public boolean onOptionsItemSelected(MenuItem item){
    switch(item.getItemId()){
        case ITEM1:
            setTitle("Start");
            break;
        case ITEM2:
            setTitle("Stop");
            break;
     }
}

Android资源概述.

Jun 24th, 2011

写代码遇到了问题,饿,本以为只是一个简单的熟悉API的过程,结果还是遇到了这么多问题…饿.继续看看教材吧 ,不要贪图快… Android中的资源访问方式大概分为两种,分别存在asset和res文件夹中.在asset中一般保存的是原生的文件,Android程序不能直接访问到,必须通过AssetManager类以二进制流的形式来读取.而res中的资源可以通过R资源类来直接访问… res中又包含anim,drawable,layout,menu,raw,values,xml文件夹.分别用来保存动画,图片,布局文件,菜单,原生文件,常量值和xml文件.
