今天看了下Linux自动化管理.一直以来都是羡慕那些系统管理员.写写脚本.研究研究其他的.控制着很大的集群.很喜欢那样的感觉.一切在自己控制中的感觉.不过数据结构老师说了.人不可能按自己所想的轨迹生活.所以.还是强迫下自己比较好.比如说我最近的厌烦实验室的心态. 扯远了.只不过是有些感慨.很羡慕那些一天对着计算机入魔的极客态度.又扯了.fuck. 定制自己的命令提示符很有用.可以输出那些你想要的信息.所以还是比较有帮助的 . 尤其是在管理一些集群的时候. 自己原来定制的PS1是:

PS1='\u->'

很简单.但是很麻烦.毕竟不能显示当前目录.而且对当前的用户也显示的不是很好. 今天看到了独特的两行模式的命令提示符号.感觉很不错.于是便有:

if [ $UID -eq 0 ];then
     SYM='#'
else
     SYM='$'
fi
DATE=`date +%D`
PS1='\n$SYM-\u@\h $DATE \T \w Bash Version:\v\n$SYM-($?)->'

实际效果还是很理想的. 下面是PS1的一些选项解释. \d 显示日期.不过因为系统的locale.所以显示格式比较垃圾.所以我自己通过date命令构建自己比较喜欢的方式. \H \h 主机名 \u 用户名 \t \T \A 显示时间. \w \W 显示当前目录. \v 显示bash版本 上面那些就够了.其实借助环境变量以及shell命令.可以得到你所要的一切·

Bash学习(四).信号

Mar 13th, 2011

Bash和Perl一样.都有着相对完善的信号处理机制.Bash是通过trap来实现,而Perl是通过%SIG来实现的. 今天说下Bash的trap实现. 先看一个简单的例子.

trap "echo 'u press ^c'" INT
trap "echo 'u press ^z';exit;" TERM
while true; do
     sleep 60
done

运行下例子就可以很容易的理解. trap的语法为:

trap cmd sig1 sig2 sig3

切记对信号的处理要放在处理代码之前. 有时候我们不想对信号进行任何处理.甚至不想让系统对该信号做出响应.我们同样是通过trap实现的.

trap "" INT TERM
while true; do
     sleep 60
done

数据结构作业.链表倒置.

Mar 11th, 2011

太窝火了.网速这么慢.连NBA都不能看.把数据结构的作业做了吧. ListNode.h

#ifndef __LISTNODE_H__
#define __LISTNODE_H__
typedef struct ListNode {
    int num;
    struct ListNode *pNext;
}Node;

typedef Node* pNode;
//初始化链表
pNode init();
//获得链表尾
pNode getTail(pNode Head);
//倒置链表
void converse_core(pNode p, pNode next);
int converse(pNode Head);
//Status:0成功.1失败.
#endif

ListNode.c

#include "ListNode.h"
#include "stdlib.h"
//初始化
pNode init(){
    int i;
    pNode Head = (pNode)malloc(sizeof(Node));
    Head->pNext = 0;

    pNode pLast,pCur;
    pLast = Head;

    for(i=0;i<10;i++){
        pCur = (pNode)malloc(sizeof(Node));
        pCur->num = i;
        pLast->pNext = pCur;
        pLast = pCur;
    }
    pCur->pNext = 0;
    return Head;
}
//获得链表尾
pNode getTail(pNode Head){
    pNode pCur = Head->pNext;
    while(pCur && pCur->pNext){
        pCur = pCur->pNext;
    }
    return pCur;
}
//转置核心递归
void converse_core(pNode p, pNode next){
    if(next){
        converse_core(next, next->pNext);
        next->pNext = p;
        return ;
    }else{
        return ;
    }
}
//转置函数
int converse(pNode Head){
    pNode pHead = Head->pNext;
    if(!pHead) return 1;
    pNode pTail = getTail(Head);
    converse_core(pHead,pHead->pNext);
    Head->pNext = pTail;
    pHead->pNext = 0;
    return 0;
}
