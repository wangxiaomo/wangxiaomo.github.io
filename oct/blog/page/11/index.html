《1988》首次出现大面积感慨 ，1988也首次出现了一个正面人物丁丁（虽然他偷车了）——在此前，韩寒小说的人物都是无所事事的。 　　其实很明显的，丁丁死于那一年。1989，我们就不多说了。 　　“我想和这个世界谈谈”，丁丁去谈了，事实表明不是因为谈崩了被崩，而是人家根本不跟你谈就把你崩了。 　　另外，对于那年，我持之以恒一个观点：不是对手太强大，而是CL太无知。 　　娜娜，她的理想是能在桑拿处工作，她的工作是销售自己的身体。我没有被她的单纯感动，也没有被她死命的要把孩子生下来感动，只是当她存满了两万块（生孩子用）就被罚走、被城管QJ、那个孙老板没给钱她死命的哭泣（她觉得有一种东西要留给她未来的老公的）那些片段让我揪心。社会很罪恶！刘茵茵，陆子野的初恋，对于这个人，没太多好说的。当陆子野（主人公）和娜娜说他和刘茵茵越好十年后在路的尽头见面，娜娜笑死掉那段蛮有感触。比较感人的是十年后陆子野真的去了，可是刘茵茵死了。 　　孟孟，这是我最喜欢的一个人物，超过丁丁。成熟的女人，有着野心的女人，她想成为明星，但是最后被潜规则，并且没红。当我看到她把陆子野发给她的冷漠的短信一句句的写在笔记本上（她的手机只能存200条短信）、陆子野一向冷冷的心被感动了给她买了个最好的手机她跑到最高的山上给陆子野打电话，我承认被感动了。还有，制片人把她潜规则了并且录像，制片人把录像拿给陆子野，陆子野别过头说：“那你怎么跟她解释”那一句，我看到了一个男人的克制与无力。想说的很多，可能是太久没看书了，所以激动，激动到这，热情退却了，不想再谈里面的人物。 　　很多人可能忽视了一点：这本书里面的每个人都有理想，最后都以惨剧收场。很多人看到了那些暗喻（很敏感，不说），却忽视了一个细节，就是10号和他女朋友的短信：我就像这个世界，这个世界是不可能被改变的。10号说看了一个电影所以写了这句话。 　　用两个词形容《1988》——悲悯+无力。 　　其实这部电影是杨德昌的《少年杀人事件》，如果把这个暗喻也理解进去，这部小说就更丰满了。 　　这部电影拍摄两蒋背景下的台湾，长达三个多小时。当时每个人都很茫然、对时什么局、命运无力，说那句话的人小明是最典型的一个，她单亲家庭，渴望得到保护，所以处处和男人勾搭，最后造就很很多人的悲剧。不是人不想去努力，是现实太强大。就像那个经典的“温水煮青蛙”理论——实际情况是青蛙在水有一定温度时候会跳出来，但是现实就是那个锅盖，在它要跳出来时盖上、把青蛙煮死——而谁是那个拿锅盖的人？ 　　 　　后记：这是我观完此书的感想，马上写了下来。结局的话，我觉得还是比较赞的。我个人觉得这本1988我想和这个世界谈谈，可以在未来10年成为韩寒作品里算得上排名的一本经典。书的内容在20年后，希望内容是过时的。真的不希望到那时，这本书里其实真实存在的故事依然在生活里真实地上演。 　　 　　最后希望今后的孩子们，都是听着娜娜式在录音笔里的摇篮曲，而不是那句“开工了”。

#!/usr/bin/perl
use utf8;
use Encode;
use LWP::Simple;

my $board_file = "board.txt";
my $recommand_file = "recommand.txt";

my $url = "http://www.ytxsj.cn/";
my $content = get($url);
my @lines = split("\n", $content);
&getRecommand(\@lines);
&getBoard(\@lines);

sub getBoard($){
    my $ref = $_[0];
    #print scalar(@$ref);
    my ($count, $i, $loopNum ) = (0, 0, scalar(@$ref));
    while($i<$loopNum){
        next unless $$ref[$i++] =~ /影片票房排行榜/;
        #print $i;
        $i += 4;
        open F,">:encoding(gb2312)",$board_file;
        while($count<5){
            next unless $$ref[$i++] =~ /(?:.*)\>(.*?)\<\/a\>/;
            $count++;
            print F $1,"\n";
        }
        close F;
    }
}

sub getRecommand($){
    my $ref = $_[0];
    #print scalar(@$ref);
    my ($count, $i, $loopNum ) = (0, 0, scalar(@$ref));
    while($i<$loopNum){
        next unless $$ref[$i++] =~ /影片推荐/;
        #print $i;
        $i += 2;
        while($count++<4){
            $$ref[$i++] =~ /src=\"(.*?)\"/;
            open F,">","$count.jpg";
            binmode F;
            print F get($1);
            close F;
        }
        #print $i;
        $i += 5;
        open F,">:encoding(gb2312)",$recommand_file;
        $count = 0;
        while($count++<4){
            $$ref[$i++] =~ /(?:.*)\>片名：\s+(.*?)\<\/li\>/;
            print F $1,"\n";
        }
        close F;
        last;
    }
}

#!/usr/bin/python
# -*- coding: utf-8 -*-

from BaseHTTPServer import HTTPServer,BaseHTTPRequestHandler

class MyHandler(BaseHTTPRequestHandler):
    def _getServerVersion(self):
        return self.server_version
    def do_GET(self):
        self.send_response(200)
        self.end_headers()
        self.wfile.write(self._getServerVersion())

server_addr = ("", 8080)
MyServer = HTTPServer(server_addr, MyHandler)
MyServer.serve_forever()
