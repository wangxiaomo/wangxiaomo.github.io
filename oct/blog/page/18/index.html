Perl ~~运算符的使用.

Mar 29th, 2011

之前在锚定的时候遇到过这个符号.和=~是相反的意思.就是无匹配为真…今天在看Effective Perl的时候发现这个东西还真是有用… 首先.当我们在检查hash数组key的存在与否和数组是否存在时使用的是:

my %hash = (
         name => 'wxm',
         score => 99,
);
if( exists($hash{name})){
      print "Exists\n";
}
my @array = (1, 2);
if( grep{ $_ == 2} @array) {
     print "Exists\n";
}

但是这样是很麻烦的.无论你是通过循环还是通过grep等方式进行判断.都是比较繁琐的..所以我们可以使用~~操作符.

print "Exists\n" if 'name'~~%hash;
print "Exists\n" if 2~~@array;

这样就相对简单了. 当然还可以直接比较数组.

# 比较数组.
print "OK" if @arrayA ~~ @arrayB;
# 比较哈希
print "OK" if %hashA ~~ @hashB;

Perl类的两种实现.

Mar 28th, 2011

之前每每看到Perl类的实现的时候总是跳过.总觉得没什么太大用处.因为对Perl功能进行过多的封装感觉有点鸡肋.不过会写还是必须的.所以就总结下. Perl类常用的两种书写形式.

# type 1.
package Person;
sub new{
    $class = shift;
    $person = {
        _id      =>   1,
        _name =>  'wxm'
    };
    bless($person,$class);
   
    sub get_Info{
        $this = shift;
        print $this->{_id} . $this->{_name} . "\n";
    }
}

这是第一种写法.也是最像其他语言的..

# type 2
package Person;
sub new{
    $class = shift;
    $person = {
         _id      =>   1,
         _name =>   'wxm'
    };
    bless($person, $class);
    return $person;
}
sub get_Info{
    $this = shift;
    print $this->{_id} . $this->{_name} . "\n";
}

上面两者.随意 `

if($0 == __FILE__){
     die "This is run as Module.\n";
     return -1;
}

Perl读取文件列表(glob)

Mar 27th, 2011

都知道使用glob(‘*’)可以获得当前路径的文件列表.可是书写代码的时候还是有些问题需要注意的.

while( defined( $file=glob('*'))){
     print "$file\n";
}

不过这里存在一个问题.就是当一个文件或者目录名为0或类似的东西时.循环就会终止.所以我们加入defined进行测试.同理.

while( defined( $line = ))){
     print "$line\n";
}

可以通过

@files = glob('*');

获得文件列表的数组.

Apache目录映射

Mar 25th, 2011

今天在实验室的台式机上重新搭建了下基本的服务.毕竟自己的笔记本还是不舍得当服务器一直开的.今天的服务配置的非常顺利.就是有个别细节我不是很懂.之前都没有仔细的研究过. 关于Apache的目录映射.之前一直以为是在sites-available里面增加配置文件.然后通过链接添加到sites-enabled中.但是今天安装phpmyadmin和torrentflux的时候发现没有关于这两个程序的配置文件.这就奇怪了.到底在哪呢? 我就谷歌了半天.最后发现应该是Apache的目录映射的原因.但是映射写在哪里了?

which phpmyadmin
#无结果.
whereis phpmyadmin
#/etc/phpmyadmin /usr/share/phpmyadmin

不用说了./usr/share/phpmyadmin下肯定是phpmyadmin的网页路径.所以直接奔到/etc/phpmyadmin下.然后我发现了一个phpmyadmin.conf文件.打开一看是Apache配置格式. 所以.肯定在apache2.2中的某一位置引用了这个配置文件.所以又回去寻找.发现在conf.d中有这个配置文件的软链接.所以一切搞定. 接下来说下目录映射的写法.

#假设别名为wp.虚拟路径为/usr/share/wordpress
Alias /wp /usr/share/wordpress

     Options Indexes FollowSymLinks
     AllowOverride None
     Order allow,deny
     Allow from all

搞定.

Apache在线离线模式.

Mar 25th, 2011

昨天在windows上又重新搭建了一个torrentflux.为了节约时间.使用的是wamp.一切搭建就绪后.发现网站只能本地访问.局域网是无法访问的.上网一查才知道wamp默认情况下是离线模式的.即只能本机访问.禁止局域网访问.为了安全?yes. 不管怎样.设置wamp为online模式.然后测试.不行.看来我还是自己手动改wamp/bin/apache/conf/httpd.conf吧. 在站点目录的配置文件中..有order allow,deny行.这里就是设置访问权限的…所以我们直接改这里..

#默认情况下.
#order deny,allow
#deny from all
#allow from 127.0.0.1
#我们修改为.
#order allow,deny
#allow from all

这样就应该可以了.需要注意的是.网上说把deny from all改成deny from none.然后把allow from 127.0.0.1 改成allow from all.不该便order deny,allow的顺序.我试着做了下.不行.所以以后的时候多试几次吧…

Wget Manual学习.

Mar 22nd, 2011

继续上篇日志.都说了.网上的那群人写的东西不是错的.就是最简单的东西.所以我决定看看wget的manual.来挖掘下社交网络的主角是怎么用–post-data这个参数的. 首先.

man wget
# Wget - The non-interactive network downloader
# 非交互式下载工具.

wget支持许多种协议.HTTP,HTTPS,FTP,以及支持代理的上述协议.这样已经够了.夫付何求?wget是非交互式下载工具.非交互的理解就是可以在后台执行,尽管用户没有log on.&可以使程序后台执行.但是必须保持SHELL开启.也就是说.用户登出后就会停止运行.所以需要配合nohup来使用.

nohup command &

这样就可已保证是non-interactive的进程了. wget可以跟踪HTML,XHTML,CSS中的链接地址.进而来产生本地镜像.wget默认情况下遵守robot.但是可以通过-e来跳过. 语法介绍: wget是通过GNU getopt来获取参数的.所以可以支持长参数也可以支持短参数.

wget -r --tries=10 http://fly.srk.fer.hr/ -o log
# 递归下载网页.每次链接至多重试10次.并把日志输出到log文件中.

-o log也可以写成-olog. wget可以设置不下载的目录.可以写在/etc/wgetrc也可以写在~/.wgetrc中.当然也可以通过参数-X来指定.

wget -X '' -X /~nobody,/~somebody
# -X ''可以指定跳过wgetrc的限制.
# 然后再通过-X来指定.
# 多个目录之间用,分割
